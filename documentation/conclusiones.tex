\section{Conclusiones}
Como conclusión, se logró implentar de manera exitosa un sistema de gestión de asistencia utilizando PostgreSQL como sistema gestor de bases de datos. A lo largo del desarrollo del proyecto, se aplicaron diversos conceptos fundamentales de bases de datos, tales como el diseño relacional, la normalización, la creación de índices para optimizar consultas, y la implementación de procedimientos almacenados y triggers para automatizar tareas y mantener la integridad de los datos.
Sin embargo también hubo problemas, ya que no se pudo comprobar que la restauración de los respaldos funcionara correctamente, debido a que el contenedor de PostgreSQL no tenía acceso a los archivos de respaldo generados por el script de backup. Pienso que se debio a alguna configuracion del volumen de ubuntu.
